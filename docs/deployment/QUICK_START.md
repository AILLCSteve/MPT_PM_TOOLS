# Quick Start Guide - PM Tools Suite

Get your PM Tools Suite running in 10 minutes!

## üìã Prerequisites

- Python 3.9+ installed
- Git installed (for deployment)
- Text editor or IDE

## üöÄ Local Setup (5 minutes)

### 1. Install Dependencies

```bash
# Create virtual environment
python -m venv venv

# Activate it
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# Install requirements
pip install -r requirements.txt
```

### 2. Configure Environment

```bash
# Copy environment template
cp .env.example .env

# Edit .env file (optional for local testing)
# The defaults work fine for local development
```

### 3. Run Application

```bash
python app.py
```

You should see:
```
INFO - PM Tools Suite - Starting Server
INFO - Server: http://0.0.0.0:5000
INFO - Available Tools:
INFO -   - CIPP Analyzer: /cipp-analyzer
INFO -   - Progress Estimator: /progress-estimator
```

### 4. Test in Browser

Open your browser to:
- **Landing Page**: http://localhost:5000
- **Health Check**: http://localhost:5000/health
- **CIPP Analyzer**: http://localhost:5000/cipp-analyzer
- **Progress Estimator**: http://localhost:5000/progress-estimator

## üåê Deploy to Render (10 minutes)

### 1. Create Git Repository

```bash
# Initialize git
git init

# Add files
git add .

# Commit
git commit -m "Initial commit"

# Create GitHub repo and push
# (Create empty repo on GitHub first)
git remote add origin https://github.com/yourusername/pm-tools-suite.git
git branch -M main
git push -u origin main
```

### 2. Deploy on Render

1. Go to [render.com](https://render.com) and sign up/login
2. Click **"New +"** ‚Üí **"Web Service"**
3. Connect your GitHub repository
4. Render auto-detects configuration from `render.yaml`
5. Click **"Create Web Service"**
6. Wait 2-3 minutes for deployment
7. Visit your app at: `https://your-app-name.onrender.com`

### 3. Set Environment Variables (in Render Dashboard)

| Variable | Value | Notes |
|----------|-------|-------|
| `SECRET_KEY` | Click "Generate" | Auto-generated by Render |
| `DEBUG` | `false` | Production mode |
| `LOG_LEVEL` | `INFO` | Standard logging |

That's it! Your app is live! üéâ


## üîß Troubleshooting

### Port Already in Use
```bash
# Use different port
PORT=8000 python app.py
```

### Module Not Found
```bash
# Make sure virtual environment is activated
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# Reinstall dependencies
pip install -r requirements.txt
```

### Can't Access Locally
- Check firewall settings
- Try 127.0.0.1:5000 instead of localhost:5000
- Verify Python is running without errors

### Render Deployment Failed
- Check build logs in Render dashboard
- Verify `requirements.txt` is correct
- Ensure Python version matches `runtime.txt`
- Check environment variables are set
